#求解点导线距离
#
##输入
  >三点坐标系，例：
  >
  >(0,0)、(1,3)、(2,3)
  >
  >以下假设为(x1,y1)、(x2,y2)、(x3、y3)

##输出
>第三点到前两点所在直线上的坐标
>
